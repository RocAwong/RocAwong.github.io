一个开始

Oct 27, 2017

为什么我选择hexo？

  从去年看到博斯的<尘世乐园>一作起，就无可遏止地遁入文艺复兴艺术史的补习中，再往后又向前向后两百年追溯了西方文艺史，无奈一直没有找到合适的载体来做一些笔记和思维拓展。之前有试过在纸质笔记本上纪录文字，然而由于我的汉字书写能力已经在工科作业的公式堆叠中渐次消亡，写出来的笔记不忍直视，也难以做长久的保存。

  之前在其他人的博客上就了解过 hexo + github 的种种益处，今晚突发奇想，在 PhiHe 的帮助下勉强搭建了自己的个人博客，以后大概会定时做一些更新，偶尔有话想说可能也会放上来。

  如果有人留意，也算是对我的一个督促吧。

  Thanks.
